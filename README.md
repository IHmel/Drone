# Drone
Реализация функционирующей модели дрона с дополнительными возможностями.
# 1 этап(сборка)
При сборке технической части дрона были использованы:
- Рама F450;
- 4 регулятора оборотов;
- 4 мотора;
- Микроконтроллер;
- Плата гироскопа;
- Передатчик с пульта управления;
- Передатчик видеосигнала
-
